## ✨ feat: add tray/overlay icons and notification settings

### English (en)
```
This update introduces tray and overlay icon support for Windows, including notification dot indicators for SVN and app updates. It adds new options to the settings:
- "Auto-start SVNTool"
- "Show window notifications"

These settings are fully localized. The main process is refactored to manage tray and overlay icon states, and all UI references to `icon.png` have been updated to `logo.png`. Additionally, dependencies are updated and notification handling for SVN and app updates has been improved.
```

---

### 日本語 (ja)
```
このアップデートでは、Windows向けにトレイアイコンとオーバーレイアイコンのサポートが追加され、SVNおよびアプリのアップデートに関する通知ドットが表示されるようになりました。設定に以下のオプションが追加されました：
- 「SVNTool を自動起動する」
- 「ウィンドウ通知を表示する」

これらの設定は完全にローカライズされています。メインプロセスはトレイおよびオーバーレイの状態管理のためにリファクタリングされ、すべてのUI参照が `icon.png` から `logo.png` に更新されました。また、依存関係が更新され、SVNおよびアプリ更新時の通知処理が改善されました。
```
